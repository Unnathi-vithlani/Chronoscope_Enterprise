BEFORE ANYTHING — DO THIS FIRST (2 minutes)
─────────────────────────────────────────────
1. Go to https://abuseipdb.com → Login → API tab → Regenerate your key
2. Copy the new key
3. Open the .env.example file in this folder
4. Rename it to .env
5. Replace YOUR_ABUSEIPDB_KEY_HERE with your new key
6. Save

NOTE: .env is in .gitignore — it will NEVER be pushed to GitHub.
      This is intentional and correct. Keep your key private.


STEP 1 — MOVE FILES TO YOUR COMPUTER
──────────────────────────────────────
1. Extract this zip folder anywhere on your computer
   Suggested location: C:\Users\unnat\chronoscope

2. Open PowerShell and navigate there:
   cd C:\Users\unnat\chronoscope


STEP 2 — SET UP GITHUB REPO
─────────────────────────────
1. Go to https://github.com/Unnathi-vithlani
2. Click "New repository"
3. Name it: chronoscope
4. Set to PUBLIC
5. Do NOT check "Add README" (we already have one)
6. Click "Create repository"

Now in PowerShell, run these commands ONE BY ONE:

   git init
   git add .
   git status

   ← Look at the list. Make sure .env is NOT in the list.
   ← If .env appears, stop and message me before continuing.

   git commit -m "CHRONOSCOPE Enterprise v2.0.0 - Real-Time Criminal Behavior Observatory"
   git branch -M main
   git remote add origin https://github.com/Unnathi-vithlani/chronoscope.git
   git push -u origin main

   ← It will ask for your GitHub username and password.
   ← Use your GitHub username + a Personal Access Token (not your password).
   ← Get token at: https://github.com/settings/tokens → Generate new token → check "repo"


STEP 3 — DEPLOY TO RENDER.COM
───────────────────────────────
1. Go to https://dashboard.render.com
2. Sign up / log in with your GitHub account
3. Click "New +" → "Web Service"
4. Click "Connect" next to: Unnathi-vithlani/chronoscope
5. Fill in:
   - Name:        chronoscope-enterprise
   - Environment: Docker
   - Branch:      main
   - Plan:        Free
6. Scroll down to "Environment Variables" → Add:
   - Key: DEMO_MODE    Value: True
   - Key: ENVIRONMENT  Value: production
7. Click "Create Web Service"
8. Wait 3–5 minutes for build to complete

Your live URL will be:
https://chronoscope-enterprise.onrender.com

IMPORTANT NOTES
────────────────
- Render free tier sleeps after 15 min of no traffic
  → First load after sleep takes ~30 seconds — this is normal
  → For demos/presentations, open the URL 2 minutes early

- Demo mode is ON by default (DEMO_MODE=True)
  → Shows 5 pre-profiled threat actors immediately
  → No API key needed for demo mode to work

- To switch to live AbuseIPDB data:
  → In Render dashboard → Environment Variables → Change DEMO_MODE to False
  → Add ABUSEIPDB_KEY with your key
  → Redeploy
